

# Caso #3 - Prevendo a Qualidade de Veículos
**Fev/21**


#### Demanda
- Uma revenda de carros quer poder prever a qualidade do carro a partir de dados de carros anteriores, e assim precificar de forma mais adequada o serviço de assistência 24 horas.
- A empresa possui uma grande base de dados de veículos adquiridos anteriormente, com características como exigência de manutenção, portas, tamanho do porta malas, segurança, etc.

O projeto deve ser desenvolvido em duas etapas, a primeira utilizando apenas o R (core) para verificar se o modelo tem performance razoável, a segunda etapa será o desenvolvimento web com Shiny.


#### Técnica utilizada
- Machine Learning, técnica de classificação.

#### Fonte de dados
- car.data (1728 instâncias e 7 atributos)
